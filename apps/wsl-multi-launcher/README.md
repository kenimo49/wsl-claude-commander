# wsl-multi-launcher

WSL (Ubuntu 24.04) を指定して、特定のディスプレイに複数のウィンドウをグリッド配置するツール。

## 概要

- **言語**: Rust
- **実行環境**: WSL側（ウィンドウ操作はPowerShell経由）
- **設定ファイル**: YAML
- **配置方法**: グリッド分割

## 機能

- 指定したWSLディストリビューション（Ubuntu-24.04）でターミナルを起動
- 特定のディスプレイを指定して配置
- グリッド分割（2x4, 3x3など）で自動配置
- 各ウィンドウで異なるコマンド・作業ディレクトリを指定可能

## 設定ファイル例

```yaml
# config.yaml
wsl_distribution: Ubuntu-24.04
target_display: 1  # 0: プライマリ, 1: セカンダリ...

layout:
  grid: "2x4"  # 2列4行 = 8ウィンドウ

windows:
  - name: "claude-1"
    command: "claude"
    working_dir: "~/workspace/project1"
  - name: "claude-2"
    command: "claude"
    working_dir: "~/workspace/project2"
```
