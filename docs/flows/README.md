## フロードキュメントの読み方

1. **入口**: 処理のトリガーとなるファイル・関数
2. **呼び出し順序**: 時系列での関数呼び出し（ファイルパス+行番号付き）
3. **シーケンス図**: Mermaidによる視覚化
4. **関連ファイル**: 処理に関わるすべてのファイル
5. **観測点**: デバッグ時のブレークポイント推奨箇所
